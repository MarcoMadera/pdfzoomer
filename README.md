## Installation

To install PDF Zoomer, run the following command:

```bash
pip install pdfzoomer
```

## Usage

To use PDF Zoomer, run the following command:


```bash
pdfzoomer -i input.pdf -o output.pdf -s scale_factor
```

Arguments:
- `-i`: Input PDF file
- `-o`: Output PDF file
- `-s`: Scale factor (optional, default=1)
